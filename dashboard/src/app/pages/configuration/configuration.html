Setup Guidance ({{ selectedOS | uppercase }})

@if (selectedOS === 'windows') {
  1. Execute the downloaded installer agent-windows.exe on the target device.
  2. During installation, all required files will be extracted to C:\Program Files\SysPulse.
  3. The installer will automatically register and start the SysPulse Monitoring Service.
  4. Once running, the agent automatically registers with the backend and begins streaming real-time statistics (CPU, Memory, Disk, Network).
  5. Open the Admin Console to see the device status change to Online.
} @else if (selectedOS === 'linux') {
  1. Transfer the downloaded agent-linux binary to the target Linux machine.
  2. Make it executable and extract to system path:
    chmod +x agent-linux && sudo mv agent-linux /usr/local/bin/syspulse
  3. Run installer command to register systemd service daemon automatically:
    sudo /usr/local/bin/syspulse --install-service
  4. The service will start automatically and immediately begin sending system statistics to the backend service.
} @else if (selectedOS === 'mac') {
  1. Double-click the downloaded agent-macos.pkg installer.
  2. Follow the package installation steps to extract files to /usr/local/share/syspulse.
  3. The installer automatically registers a LaunchDaemon service to run on startup.
  4. The background agent service starts collecting and saving metrics immediately.
}
